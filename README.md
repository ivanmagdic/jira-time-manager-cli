## Installation

```shell

```

## Usage

### Initialization

```shell
jtm init --url <public-jira-url> -u <username> -p <password>
```

### List issues

```shell
jtm list-issues
```

### Start progress on issue

```shell
jtm start <issue-id>
```

### Stop progress on currently active issue
```shell
jtm stop
```
